# GesturePlay: Smart Touchless Media Control System

Experience the future of media interaction. **GesturePlay** offers an intuitive, touchless way to interact with your digital content, completely redefining user convenience, accessibility, and hygiene. By transforming real-time camera feeds into swift action streams, users can manage their playback systems completely hands-free.

---

## 🚀 Key Features

*   **Touchless Control:** Dynamic system execution up from a distance of up to 2 meters, ensuring fully hygienic interactions.
*   **Minimalist Light User Interface:** A gorgeous, high-contrast premium light theme complete with custom-designed glassmorphic and fully transparent components.
*   **Privacy-Centric Architecture:** All video frames are processed entirely on-device, offering rapid response times and secure local data sandboxing.
*   **Multi-Format Control Hub:** Clean, responsive structural grids separating dedicated workflows for Audio, Ultra-HD Video, and Live Testing/Diagnostics routines.

---

## 🧠 Active Gesture Reference Deck

Operate your media system using high-accuracy custom hand movements captured via any high-resolution ($720\text{p}+$) camera terminal:

| Gesture Trigger | Target System Action | Operational Behavior |
| :--- | :--- | :--- |
| **Right Hand Up** | Play / Resume | Instantly fires a command sequence to start or resume active media playback. |
| **Left Hand Up** | Pause | Halts active frame or track streaming seamlessly across view panes. |
| **Hands Crossed** | Rewind / Skip | Executes a backward timeline skip or fast-rewind playback loop. |
| **Enmaten Sign** | Max Volume | Instantly shifts output amplification structures to higher maximum thresholds. |

---

## 🛠️ System Architecture Execution Flow
1. **Camera Input:** High-resolution live capture streams coordinate spatial vector boundaries.
2. **ML Processing:** Matrix indices run locally inside a client-side machine learning engine (powered by Google's Teachable Machine framework).
3. **API Integration:** Verified commands translate instantly into media application events via accessibility framework APIs.
4. **Visual Feedback:** Heads-up interface elements render instant responses back to the operator.

---

## 💻 Tech Stack & Framework Dependencies

*   **Frontend:** Semantic HTML5, CSS3 Custom Properties (featuring Glassmorphism overlays and fully transparent styling modules), JavaScript (ES6 Modules).
*   **Machine Learning Core:** Supervised Machine Learning Algorithms via Google’s Teachable Machine platform.
*   **Typography:** Google Fonts Integration (`Montserrat`, `Open Sans`, `Economica`).
